# AI Interview Coach
AI Interview Coach - An AI-powered interview preparation tool that generates interview questions and evaluates answers using Gemini AI.

## Overview
AI Interview Coach is an AI-powered interview preparation tool that generates interview questions and evaluates answers using Google's Gemini AI.

## Features
- Role-based interview questions
- AI-powered answer evaluation
- Score and feedback generation
- Improved answer suggestions
- Streamlit web interface

## Technologies Used
- Python
- Streamlit
- Google Gemini API
- GitHub
- Streamlit Cloud

## Workflow
1. User enters role
2. AI generates interview questions
3. User submits answer
4. Gemini evaluates answer
5. Feedback is displayed

## Future Enhancements
- Resume PDF upload
- Resume-based questions
- Mock interview mode
- Voice-based interviews
- Performance analytics
